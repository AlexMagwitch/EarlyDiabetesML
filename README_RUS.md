# EarlyDiabetesML

Учебно-исследовательский проект по предсказанию риска диабета на ранних стадиях на основе анкетных данных (симптомы и базовая информация о пациенте).

Репозиторий содержит код, датасет и материалы выпускной квалификационной работы (thesis).

---

## Обзор проекта

Цель проекта – проверить, достаточно ли данных простого опросника (жалобы, симптомы, возраст, пол и т. д.), чтобы обучить модель, способную предсказывать наличие диабета на ранней стадии.

В репозитории находятся:

- `early_diabetes_ml.py` – основной скрипт с анализом данных и моделями машинного обучения
- `diabetes_data.csv` – исходный датасет (разделитель `;`)
- `EarlyDiabetesML_Thesis.pdf` – текст выпускной квалификационной работы (публичная версия)
- `presentation.pptx` – презентация к защите
- `requirements.txt` – список зависимостей Python

Во время работы скрипта дополнительно создается файл `describe.csv` с описательной статистикой. Этот файл не хранится в репозитории и добавлен в `.gitignore`.

---

## Используемые модели

В проекте реализовано обучение и сравнение нескольких классических моделей классификации:

- Logistic Regression (логистическая регрессия)
- k-Nearest Neighbors (k ближайших соседей, kNN)
- Decision Tree (решающее дерево)
- Random Forest (случайный лес)

Для каждой модели выводятся метрики качества:

- Accuracy
- Recall
- Precision
- F1-score

Также скрипт строит несколько визуализаций:

- распределение целевого признака (наличие или отсутствие диабета);
- распределения по полу, возрасту и симптомам;
- матрицу корреляций признаков (heatmap).
